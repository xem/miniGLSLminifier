Mini GLSL minifier (WebGL 1.0 / 2.0)

- Removes comments
- Removes unnecessary spaces, tabs, line breaks
- Removes unnecessary zeroes in numbers (floats, ints, octals, hex)
- Inserts "\n" around #macros to keep all the code on a single line
